This extention is developed to get ontime analysis for data shown in visible table for one page as of now. 

To add this into chrome follow the steps:
1. Open Google Chrome
2. Click on extention from toolbar on right side of address bar, select manage extentions
3. Enable developer mode from top right corner 
4. From top left corner, select load unpacked. Browse and select the un-zipped extention folder
5. Pin down the extention for ease of use

To use the extention:
1. Visit url: https://ontime.armakuni.in/timesheet or https://ontime.simformsolutions.com/timesheet
2. Click on pinned extention and push "Get Ontime Matrix" button
3. Ontime analysis matrix will be showed just below the heading "Timesheet".
